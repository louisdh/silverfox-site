Terminal for iOS

Saturday, 6 January 2018

by Louis D'hauwe

tags: blog, ipad, open source, panelkit, swift, terminal

Today I’m releasing Terminal, a sandboxed command line interface, for iOS. The app is free, with no in-app purchases or ads. It’s available on the iOS App Store for iPhone and iPad.

This project would not have been possible without open source software. Therefore it’s only logical to make the app open source as well. It’s on GitHub and contributions are welcome.

As a nice bonus I have implemented PanelKit for the commands history.
