The Cub Programming Language & SavannaKit

Sunday, 4 February 2018

by Louis D'hauwe

tags: blog, cub, lioness, open source, openterm, savannakit, swift

Today I'm releasing two new projects:

The Cub Programming Language

Cub is an interpreted scripting language inspired by Swift. It was designed specifically for scripting in OpenTerm. Cub was forked from Lioness, my (first) programming language I released past spring. As you might expect by now, Cub is open source!

SavannaKit

SavannaKit is a protocol oriented framework for creating IDEs for iOS and macOS, written in Swift. It contains a SyntaxTextView, which is a cross-platform Text View with syntax highlighting and line numbers. SavannaKit is also open source!

I first teased SavannaKit back in June. It feels so good to finally release it.
